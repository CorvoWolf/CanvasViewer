画布预览面板 Canvas Viewer Panel

该脚本为Krita提供了一个实时画布预览面板，可以在独立窗口中查看当前画布的缩略图。主要功能包括：

This script provides Krita with a real-time canvas preview panel that allows viewing thumbnails of the current canvas in a separate window. Main features include:

  • 实时预览当前画布内容 / Real-time preview of current canvas content
  • 自动适应窗口大小 / Automatic window size adjustment
  • 支持高DPI显示 / Supports high DPI display
  • 智能刷新机制，优化性能 / Smart refresh mechanism for optimized performance

使用方法 Usage

  1. 在Krita的"设置"菜单中启用该插件 / Enable the plugin in Krita's "Settings" menu
  2. 通过"设置"->"面板列表"->"画布预览"打开面板 / Open the panel via "View" -> "Dockers" -> "Canvas Viewer"
  3. 面板会自动显示当前画布的缩略图 / The panel will automatically display a thumbnail of the current canvas
  4. 可以通过拖动面板边缘调整大小 / You can resize the panel by dragging its edges

注意事项 Notes

  • 建议在性能较低的设备上适当调整刷新间隔 / It is recommended to adjust the refresh interval appropriately on lower-performance devices
  • 如果遇到显示问题，请检查DPI设置 / If you encounter display issues, please check the DPI settings
  • 面板尺寸会根据屏幕分辨率自动调整 / The panel size will automatically adjust based on screen resolution
